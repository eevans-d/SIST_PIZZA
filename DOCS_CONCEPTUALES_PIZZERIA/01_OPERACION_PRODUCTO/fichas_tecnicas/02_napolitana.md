# Ficha Técnica: Pizza Napolitana

---

### **Descripción Breve:**
Un clásico fresco y sabroso. La combinación de tomate fresco, ajo y aceitunas sobre la base de muzzarella la hace una de las favoritas.

---

### **Ingredientes (Toppings):**
- [ ] Salsa de tomate
- [ ] Muzzarella
- [ ] Tomate fresco en rodajas
- [ ] Ajo picado
- [ ] Aceitunas verdes

---

### **Instrucciones de Armado:**
1.  Esparcir una capa uniforme de **salsa de tomate** sobre la base.
2.  Cubrir con **muzzarella**.
3.  Distribuir las rodajas de **tomate fresco** de manera uniforme.
4.  Añadir el **ajo** finamente picado sobre el tomate y el queso.
5.  Colocar las **aceitunas**.

---

### **Notas de Producción:**
- Cortar el tomate en rodajas no demasiado gruesas para que se cocine bien.
- No excederse con el ajo para no opacar los otros sabores.
