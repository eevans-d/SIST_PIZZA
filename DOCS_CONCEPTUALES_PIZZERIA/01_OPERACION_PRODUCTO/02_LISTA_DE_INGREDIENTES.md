# Lista Consolidada de Ingredientes (Toppings)

Esta es la lista maestra de todos los ingredientes necesarios para el armado de las pizzas del menú. La **base de masa madre precocida** se considera un producto aparte y se gestiona con su propio proveedor.

La lista está ordenada alfabéticamente para facilitar la gestión de stock y los pedidos a proveedores.

---

- Aceitunas (verdes)
- Ajo (fresco, para picar)
- Cebolla
- Huevo (para cocer)
- Jamón (cocido, en fetas)
- Longaniza Calabresa (en rodajas)
- Morrones (asados, en tiras)
- Muzzarella (en horma o rallada)
- Parmesano (en hebras o para rallar)
- Provolone (en horma)
- Roquefort (en horma)
- Salsa de tomate (base para pizza)
- Tomate (fresco, para cortar en rodajas)
