# Plan de Diseño y Equipamiento: Local / Pizzería

Este documento es la guía maestra para la configuración del espacio de producción. La eficiencia, la higiene y la funcionalidad son los principios rectores.

---

## 1. Puesta a Punto y Distribución del Espacio (Layout)

- **Objetivo:** Diseñar una distribución que optimice el flujo de trabajo del pizzero y minimice los movimientos innecesarios.

- **Zonificación Propuesta:**
  1.  **Zona de Recepción y Cobro:** El mostrador. Donde se toman los pedidos (si hay take away) y se gestionan los pedidos de delivery.
  2.  **Zona de Almacenamiento Frío:** Heladera y/o freezer para bases, quesos, fiambres, etc.
  3.  **Zona de Armado:** La mesa de trabajo principal. Debe estar estratégicamente ubicada entre la zona de almacenamiento y el horno.
  4.  **Zona de Cocción:** El área del horno.
  5.  **Zona de Corte y Empaque:** Una superficie limpia y despejada para cortar la pizza y prepararla para su entrega.
  6.  **Zona de Lavado (Bacha):** Para la limpieza de utensilios.

- **Flujo de Trabajo Ideal:**
  `Recepción Pedido -> Búsqueda de insumos en Heladera -> Armado en Mesa de Trabajo -> Horneado -> Corte y Empaque en mostrador -> Despacho`

- **Checklist de Instalaciones:**
  - [ ] Verificar la potencia eléctrica disponible para el horno y la refrigeración.
  - [ ] Asegurar una correcta ventilación y extracción de humos para el horno.
  - [ ] Comprobar las tomas de agua y desagües para la zona de lavado.
  - [ ] Planificar suficientes enchufes en la zona del mostrador y la cocina.
