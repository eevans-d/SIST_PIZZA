# Plan de Diseño: Patio

## Concepto General
- **Objetivo:** Transformar el patio en un espacio acogedor y con identidad, que funcione como una extensión de la marca y un punto de atracción visual desde el pasillo de la galería.
- **Estilo Propuesto (Brainstorming):**
  - **Opción A: Rústico Italiano:** Madera, terracota, muchas plantas (ej. malvones, olivo en maceta), luz cálida.
  - **Opción B: Moderno Industrial:** Metal negro, cemento alisado, muebles de líneas simples, luz de neón.
  - **Opción C: Bohemio / "Chill":** Pallets, almohadones, alfombras de exterior, guirnaldas de luces tipo kermesse.

---

## Elementos Clave a Definir

### 1. Iluminación
- **Función:** Crear ambiente, especialmente en el horario pico (noche).
- **Ideas:**
  - [ ] Guirnaldas de luces cálidas (tipo Edison o kermesse) cruzando el espacio.
  - [ ] Faroles de pared en puntos estratégicos.
  - [ ] Iluminación puntual sobre plantas o elementos decorativos.
  - [ ] Velas o centros de mesa con luz LED en cada mesa (si aplica).

### 2. Mobiliario
- **Función:** Proveer un espacio de espera cómodo o para consumo rápido.
- **Ideas:**
  - [ ] Bancos largos de madera contra una pared.
  - [ ] Mesas altas tipo barra con banquetas.
  - [ ] Conjunto de 2-3 mesas bajas con sillas.
  - [ ] Muebles hechos con pallets para un look reciclado y económico.

### 3. Vegetación y Elementos Naturales
- **Función:** Aportar vida, color y frescura al espacio.
- **Ideas:**
  - [ ] Macetas grandes en las esquinas con plantas resistentes.
  - [ ] Jardín vertical en una de las paredes.
  - [ ] Pequeñas macetas en las mesas con hierbas aromáticas (romero, albahaca).

### 4. Decoración y Marca
- **Función:** Reforzar la identidad de la pizzería.
- **Ideas:**
  - [ ] Un mural pintado a mano con el logo o una temática italiana.
  - [ ] Pizarras grandes para escribir el menú, promociones o frases del día.
  - [ ] Señalética de madera o metal indicando la entrada al local.
  - [ ] Usar elementos reciclados como botellas de tomate para decorar.
